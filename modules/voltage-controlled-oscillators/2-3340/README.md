# CHAIN (CEM-3340)
(Voltage Controlled Oscillator)

#### Current Version: 1

## Inputs

| type   | value              |
|--------|--------------------|
| jack   | 1v/oct             |
| jack   | FM                 |
| jack   | Sync               |
| jack   | PWM                |
| switch | hard/soft sync     |
| knob   | frequency (course) |
| knob   | frequency (fine)   |
| knob   | PWM amount         |
| knob   | PWM control        |

## Output

| type | value |
|------|-------|
| jack | SAW   |
| jack | TRI   |
| jack | PULSE |
| jack | SINE  |
